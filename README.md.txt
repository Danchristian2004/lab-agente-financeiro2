# 🏦 B-Fin AI Advisor

Agente Financeiro Inteligente com IA Generativa desenvolvido para o desafio **Bradesco - GenAI & Dados**.

O projeto tem como objetivo criar um assistente virtual financeiro capaz de analisar dados do cliente, personalizar recomendações e apoiar decisões financeiras com segurança, clareza e foco em metas.

---

## 📌 Visão Geral

O **B-Fin AI Advisor** transforma informações financeiras em ações práticas.

A partir de dados como histórico de transações, perfil de investidor e metas financeiras, o agente oferece recomendações inteligentes e contextualizadas, atuando como um consultor financeiro digital.

---

## 🚀 Funcionalidades

- 📊 Análise de gastos e receitas
- 💰 Sugestão de investimentos compatíveis com o perfil
- 🎯 Acompanhamento de metas financeiras
- 🤖 Chat interativo com IA
- 🔒 Respostas seguras e baseadas em dados
- 📈 Apoio à tomada de decisão financeira

---

## 🧠 Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- JSON / CSV
- IA Generativa
- Engenharia de Prompt
